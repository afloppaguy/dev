# Media guide

The site does not show fake video boxes. Add a real file and then add its path to the project's `media` array in `data/projects.json`.

Recommended MP4 settings:

- H.264 video
- AAC audio
- 1280×720 or 1920×1080
- Each file below 100 MB
- Short clips where possible

The code hides a failed image or video automatically. When every media item is missing, the whole media section is removed.
